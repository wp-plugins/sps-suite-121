Suite for enhancing your 'static' pages and archives. This plugin hides pages from the page list, displays a navigation row and more.

== Description ==

SPS-Suite is the successor of "SPS -Sidebar Page Switcher". It enhances your page-list and your archive-list in the sidebar as well as your search function. This plugin is useful for blogs with many static subpages and the wish of a small sidebar.<br />
Please take a look at the screenshots or at the pages mentioned below for a better understandig of the plugin's functions.<br />
<em>Now including a widget for your widget-ready sidebar.</em><br />
There are several options implemented in the plugin:<br />
1. You can have the sidebar-page-list be more smart, e.g. only pages of the top level and all useful level are displayed, additionally you can hide pages from being displayed in the sidebar.<br />
2. You can extend your search function with a fulltext index, this search uses articles and pages and is restricted to MySQL 4.0 or higher.<br />
3. Addionally to the sidebar function this plugins allows you to display a navigation bar, which can help your users to maneuver through many sub- and subsubpages.<br />
4. Last not least this plugin shortens your archive list to a maximum of month displaying also all years.<br />
<br />
Feel free to look at <a href="http://www.tolkiengesellschaft.de>http://www.tolkiengesellschaft.de</a> or <a href="http://www.hobbingen.de">http://www.hobbingen.de</a> (German only) for a detailed picture of the functions (both pages use the plugin intensively).<br />
(Also compatible to WP 1.5 to WP 2.0  with an old version of the plugin, see plugin homepage)


== Installation ==

1. Upload `sps-suite.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure all the functions through the 'Settings' menu in WordPress
Optional: Use the "SPS - List Archive" widget in your theme


== Frequently Asked Questions ==

= How will I know that this is what I want ? =

You will know it, if you test all the functions in your blog

= Why are all these function implemented in one plugin ? =

Originally this plugin was written for the homepage of the German Tolkien society (tolkiengesellschaft.de) and all functions were not found elsewhere.

= I want the plugin in another language, are there any further translations? =

No, this plugin is in English and German only, but you may add this easily translate it. Please contact me if you want to translate this plugin.
